Card Matching Game
A classic memory-based card-guessing game where players find pairs of matching cards to score points. This project focuses on smooth gameplay logic and state management.
>

🎮 How to Play
Flip Cards: Click on any two cards to reveal their symbols.

Find Matches: If the two cards match, they stay face up, and your Score increases by 1.

Try Again: If they don't match, they will automatically flip back over after a short delay.

Track Progress: The game tracks your total number of Moves and your current Score.

Restart: Use the "Restart" button at any time to shuffle the deck and reset your stats.

✨ Features
Real-time Scoring: Instant updates to your score upon finding a pair.

Move Counter: Keeps track of how many attempts you've made.

Auto-Flip Logic: Handles non-matching pairs automatically.

Reset Functionality: Quick game reset without refreshing the page.

Responsive Design: Playable on various screen sizes.

🛠️ Tech Stack
React (Frontend Framework)

CSS3 (Styling and Flip Animations)

JavaScript (ES6+) (Game Logic)

🚀 Getting Started
Prerequisites
Node.js installed on your machine

A browser like Chrome or Firefox

Installation
Clone the repository:

Bash
git clone https://github.com/your-username/card-guessing-game.git
Navigate into the project directory:

Bash
cd card-guessing-game
Install dependencies:

Bash
npm install
Start the development server:

Bash
npm start
